# Glosario

> Función: **mejorar la recuperación.** Un recruiter puede preguntar "¿optimiza conversiones?" y un Head "¿trabaja el funnel?"; ambos deben llegar al mismo conocimiento. Este documento mapea sinónimos y equivalencias ES/EN para que el vocabulario del visitante no rompa la búsqueda semántica.
> Cada entrada: *término canónico — sinónimos/variantes — documento donde vive*.

## Producto y negocio
- **CRO** — optimización de conversión · conversion rate optimization · optimizar conversiones · mejorar el funnel → `business-thinking.es.md`, `proj-cro`.
- **Funnel** — embudo · embudo de conversión · proceso de reserva → `proj-cro`, `proj-ilunion-booking`.
- **Canal directo** — venta directa · direct channel · reserva directa → `proj-ilunion-booking`.
- **Impacto de negocio** — retorno · ROI · resultados · valor de negocio → `metrics.es.md`, `business-thinking.es.md`.
- **KPI** — métrica · indicador → `metrics.es.md`.

> Nota de desambiguación crítica para el RAG: **"IA" es ambiguo en español** — puede ser *arquitectura de información* o *inteligencia artificial*. El sistema debe resolverlo por contexto; este glosario lo marca explícitamente.

# Impacto y Resultados (RESULT)

La reorganización UX redujo la carga cognitiva, mejoró la escaneabilidad en tareas frecuentes y estableció una base sistemática para futuras evoluciones del sistema. La colaboración diseño-desarrollo fue más fluida con menor retrabajo en iteraciones posteriores.

| Área | Antes | Después |
| :--- | :--- | :--- |
| **Contraste WCAG** | Falla criterio AA en múltiples componentes | Criterio AA cumplido en todos los textos |
| **Navegación por teclado** | Elementos críticos inaccesibles sin ratón | 100% de elementos interactivos accesibles |
| **Jerarquía semántica** | Estructura heading incorrecta entre módulos | Estructura semántica correcta para lectores de pantalla |
| **Estados interactivos** | Focus / hover sin diferenciación visual | Estados claramente diferenciados en todos los componentes |
| **Consistencia entre módulos** | Patrones distintos en cada módulo sin regla unificada | Design System aplicado — patrones reutilizables |

> 📌 *Métricas cuantitativas de uso son confidenciales. Los resultados reflejan auditoría WCAG, validaciones funcionales y observación de comportamiento en el entorno académico digital.*

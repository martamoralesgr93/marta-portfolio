# Impact & Results

The UX reorganisation reduced cognitive load, improved scannability on frequent tasks and established a systematic foundation for future system evolutions. Design-development collaboration became more fluid with less rework in subsequent iterations.

| Area | Before | After |
| :--- | :--- | :--- |
| **WCAG Contrast** | Fails AA criterion on multiple components | AA criterion met on all texts |
| **Keyboard navigation** | Critical elements inaccessible without mouse | 100% of interactive elements accessible |
| **Semantic hierarchy** | Incorrect heading structure across modules | Correct semantic structure for screen readers |
| **Interactive states** | Focus / hover without visual differentiation | States clearly differentiated across all components |
| **Cross-module consistency** | Different patterns in each module, no unified rule | Design System applied — reusable patterns |

> 📌 *Quantitative usage metrics are confidential. Results reflect a WCAG audit, functional validations and behaviour observation in the academic digital environment.*
